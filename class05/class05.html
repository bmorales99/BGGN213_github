Let’s add a line

Let’s add a line and a title, subtitle and caption as well as axis labels

ggplot(cars) +
  aes(x = speed, y = dist) +
  geom_point() + 
  geom_smooth(method = "lm", se = FALSE) + 
  labs(title = "Silly Plot", x = "Speed (MPH)",
       y = "Distance (ft)",
       caption = "when is tea time anyway??") +
